# PhaserJsDemoGame
A Phaser JS Demo Game.

<h4> To Run </h4>
<ol>
<li> Have Node and Grunt installed. </li>
<li> npm install</li> 
<li> grunt </li>
<li> Play (or code with live reload)</li>
</ol>
